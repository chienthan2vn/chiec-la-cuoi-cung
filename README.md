# Hệ thống Nhận diện Bệnh trên Lá cây (Plant Disease Classification)

Dự án MLOps sử dụng PyTorch, FastAPI, Airflow, MLflow, MinIO, Docker và Google Cloud.

## Cấu trúc thư mục
Xem chi tiết tại file `treefile` hoặc `documents.md`.

## Hướng dẫn nhanh
1. Cài đặt môi trường: `pip install -r requirements.txt`
2. Chạy hạ tầng: `cd infra && docker-compose up -d`
